# ✨ Next Action Router

## Aim

> This library aims to simplify working with nextjs server actions. One can easily
> compose complex server actions using the provided abstractions.

## Features

- 🔀 Action Routing
- ⚙️ Input Validation
- 🪶 Light weight
- 🗒️ Logging
- ⛓️ Middleware Chaining
- 🔥 Robust error handling
- ⚡ Snappy DX
- 🇹 Type Safety
