## Compatibility

Current version is only tested for `nextjs v14 (app router)`. Both javascript and typescript projects written
in nextjs v14 are Compatible. Support for nextjs v13 and v15 will roll out sooner.

<Callout type="warning">
Requirements:
- Next.js ^v14
- React >= 18.2.0
- TypeScript ^v5 (optional)
- Zod ^v3

</Callout>

## Required

Since `next-action-router` uses zod for input validation.
So make sure to install zod in your project.

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tabs.Tab>

```bash copy
  npm install zod
```

  </Tabs.Tab>
    <Tabs.Tab>

```bash copy
  yarn add zod
```

  </Tabs.Tab>
  <Tabs.Tab>
 
```bash copy
  pnpm install zod
```

  </Tabs.Tab>
</Tabs>
